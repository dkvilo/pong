# The Pong

This is a simple pong game made in a few hours. I hope you enjoy it.

<img src="https://github.com/dkvilo/dk_pong/blob/master/images/prev.png" />

## Player 1 Controls

- Up: W
- Down: S

## Player 2 Controls

- I: Up Arrow
- J: Down Arrow

# Build

You will need to install the following libraries:

- Sdl2
- Sdl2_ttf

```bash
$ make clean build
$ ./build/pong
```
